# NLP Project 1.1 — Task 1 and Task 2 Handoff

This folder contains my part of the assignment:

- **Task 1: Exploratory Data Analysis**
- **Task 2: Text Preprocessing**
- Report text for Task 1 and Task 2
- Experiment summary sheet placeholder/copy

## Dataset

Please place the dataset file here:

```text
data/Sentences_50Agree.txt
```

Only this file should be used for the project, as required in the assignment.

## How to run

From this folder, run:

```bash
python src/task1_eda.py
python src/task2_preprocessing.py
```

Or run both together:

```bash
python run_task1_task2.py
```

## Outputs

The scripts create outputs in:

```text
figures/
```

## Notes for teammate

The EDA and preprocessing are designed to support later modeling tasks. The preprocessing script saves both original text and processed text, so the team can compare models with raw and preprocessed versions.
