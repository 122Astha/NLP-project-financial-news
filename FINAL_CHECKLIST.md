# Final Checking Checklist Before ZIP Submission

## Required files
- [ ] Source code is included.
- [ ] Report PDF is included.
- [ ] Experiment summary sheet is included.
- [ ] ZIP file name follows the required format: NLP_project_1_1_[Group Name].zip or NLP_project_1_1_[Surname(s)].zip

## Code check
- [ ] Code runs from a clean folder.
- [ ] Sentences_50Agree.txt is placed in data/.
- [ ] Task 1 script runs.
- [ ] Task 2 script runs.
- [ ] Task 4 script runs.
- [ ] Output files are generated in results/.
- [ ] Figures are generated in figures/.

## Report check
- [ ] Report uses ACM format.
- [ ] Report is maximum 4 pages.
- [ ] Task 1 includes interpretation, not only figures.
- [ ] Task 2 explains why each preprocessing step was used.
- [ ] Task 4 explains most similar and least similar pairs using actual output values.
- [ ] Tables and figures are readable.
- [ ] The writing sounds like your own explanation.

## Reproducibility check
- [ ] Random seed is mentioned or fixed.
- [ ] Python packages are mentioned.
- [ ] Important parameter choices are explained.
- [ ] Results in the report match generated output files.

## Plagiarism safety
- [ ] Do not copy text blindly.
- [ ] Read every paragraph and adjust it to your actual results.
- [ ] Replace generic examples with your real output values.
- [ ] Make sure the final text reflects what your code actually produced.
